# Bob Contributor Program

**For early community members who want to help build Bob — no Mac required.**

---

## The Deal

You're not a helper. You're a founder-tier contributor. The people who show up before the launch are the ones who get credit when it takes off.

Everything here is zero-cost, Windows-compatible, and meaningful.

---

## Roles

### 🛡️ Bob Ambassador
*Spread the word, grow the community*
- Share Bob content in your circles (WhatsApp, Twitter/X, Discord, Telegram)
- Recruit new members to the Discord
- Build a local Bob community group (WhatsApp group, Telegram chat, etc.)
- Represent Bob at any tech/dev events or meetups

### 📝 Bob Scribe
*Write the words that explain Bob to the world*
- Write FAQs and help docs (Google Docs, submitted to Gravy)
- Draft blog posts, Reddit launch posts, Product Hunt descriptions
- Write Discord announcements and community updates
- Caption and transcribe demo videos

### 🔍 Bob Scout
*Research tasks that feed the project*
- Domain availability checks (like HNS name lookups)
- Competitor research
- Find relevant communities, subreddits, Discord servers to post in
- Source free assets: icons, fonts, stock images
- Gather pricing data for features (what do competitors charge?)

### 🎨 Bob Designer
*Visual feedback and creative work*
- Give feedback on UI screenshots and mockups (no Mac needed — Gravy shares screens)
- Create memes, banners, social graphics (Canva, Figma free tier)
- Write short video scripts for demo reels
- Suggest copy improvements on the website and marketing materials

### 🌍 Bob Translator
*Make Bob speak every language*
- Translate UI strings into Yoruba, Igbo, Hausa, Nigerian Pidgin
- Review translations from other contributors
- Suggest culturally appropriate phrasing for Nigerian market

### 🛠️ Bob Tester (Web)
*QA anything that runs in a browser*
- Test web-based Bob tools (any browser on Windows)
- Report bugs with screenshots and steps to reproduce
- Verify the landing page and marketing site on Windows browsers
- Test the HNS registrar site when it launches
- Check mobile responsiveness on phones

### 🤝 Bob Mod
*Keep the community healthy*
- Welcome new Discord members
- Answer common questions in #help channels
- Escalate issues to Gravy when needed
- Keep conversations on track
- Enforce community rules fairly

## How Tasks Work

Tasks live in **#task-board** in Discord (pinned posts, one task per message).

Each task has:
- **What:** Clear description of what to do
- **How:** Step-by-step if needed
- **Output:** What to submit and where (Google Form, Discord thread, GitHub issue)
- **Credit:** Who gets named for completing it

To claim a task: reply "claiming" in the thread.
To submit: post your output in the same thread.

**One task at a time per person** — finish before you grab another.

Every contributor gets:
- Name in the `CONTRIBUTORS.md` file in the Bob Board GitHub repo
- Role badge in Discord matching their contributor type
- Listed on the bobbrowser.com community page (coming soon)
- First access to Windows beta when it drops
- Reserved Bob username of their choice

Top contributors each month get:
- **Bob OG** role in Discord (permanent, visible)
- Shoutout in launch materials
- Direct line to Gravy for feature suggestions

## Windows-Specific Notes for Testers

Since Bob Board is currently Mac-only, Windows testers focus on:
- **bobbrowser.com** — works on any browser
- **Bob community/marketing pages** — test on Chrome, Edge, Firefox on Windows
- **Web demos** — any browser-based tools Gravy shares
- **HNS registrar site** (when live) — pure web, works everywhere
- **Mobile testing** — test on your phones (Android, iPhone)

When Windows support drops, you'll be first in line for the real Electron app beta.
