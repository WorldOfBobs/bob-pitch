And it definitely shouldn't cost hundreds of dollars a year.

03 — Competitive Advantages
What nobody else has.
Truly local-first — nothing leaves your machine. No accounts, no telemetry, no cloud dependency. Privacy by design, not by policy.
Open source (MIT) — builds trust with the dev community. Anyone can audit, fork, or contribute. The code is yours.
Permanently free core — not a trial, not a lite tier. The 12 tools are free forever. We monetize the network, not the software.
Zero setup — one download, one launcher. No configuration, no API keys, no onboarding wizard for basic tools. Works in 30 seconds.
Replaces $300+/year of tools — Color Bob alone replaces ColorSnapper ($15). Clipboard Bob replaces Paste ($30). Drop Zone replaces several converters. Free wins.
Built for everyone, not just devs — clean UI, single hotkey access, no CLI, no scripts required. Your non-technical friends can use this.
04 — Unfair Advantages
Why we win long-term.
Distribution Moat
Open source + free = zero friction to try. Once Bob is in someone's menu bar, it becomes invisible infrastructure. They don't uninstall invisible infrastructure.
Dev Community Flywheel
Every developer who contributes a Bob app is free R&D. GitHub stars compound. Show HN posts write themselves. The open source community does our marketing for us.
Speed of Shipping
12 polished apps built and launched in weeks. Raycast has 60 employees. We move faster because we're not protecting a subscription model — we're building a platform.
First Mover — Free Launcher
Raycast went paid. Alfred has been around since 2011. Nobody has launched a modern, polished, fully free Mac launcher built with current tech. We're filling a real gap.
