03 — Competitive Advantages
What nobody else has.
Local-first — nothing leaves your machine. Verifiable. Privacy by default, not by policy.
Open source (MIT) — builds trust with dev community. Anyone can audit, fork, contribute.
Bob Net network effect — your contacts, your handle, your chat history. Switching launchers means leaving your network.
36 tools in one — replaces Raycast + CleanShot + ColorSnapper + Paste + TextExpander + more.
Agent Bob one-click AI — no setup, no API keys, works for non-technical users out of the box.
$BOB token economy — users have a financial stake in the ecosystem. No other launcher has this.
Task marketplace — intimidated users can hire other Bob users to run tasks for them. Creates jobs inside the ecosystem.

Why we win long-term.
Network Lock-in
Your Bob-handle is yours. Your Bob Net contacts know you by it. Your chat history lives there. Moving launchers means starting over socially — same reason nobody leaves WhatsApp.
Token Switching Cost
$BOB earned through relay nodes, storage lending, referrals — once accumulated, users don't want to forfeit earned value. Same flywheel as airline miles.
First Mover — Local AI Agent
Claude Computer Use API is brand new. Nobody has made it one-click for regular people. We ship it first, we own the category.
Open Source Community
Every developer who contributes a Bob app or automation template is free marketing + free R&D. The code is open. The network is ours.
